구조원리

농기계는 흐름으로 이해합니다

농기계는 하나의 부품으로 움직이지 않습니다.
전기, 동력, 유압, 조향이 서로 연결되어 흐름을 만들고,
그 흐름이 무너질 때 고장은 가장 약한 곳부터 드러납니다.

전기 → 동력 → 유압 → 조향의 큰 구조를 먼저 보고, 그 안에서 생성 → 전달 → 사용의 흐름으로 하나씩 들어가 보세요.
전체 시스템 보기 선택해서 깊게 보기

전체 시스템

먼저 큰 흐름을 봅니다. 아래 항목을 누르면 선택된 시스템이 아래에서 전면으로 올라옵니다.

전기
01

기계를 깨우고 시동하며 각 장치를 제어하는 시작점입니다.

생성 → 전달 → 사용
동력
02

엔진에서 힘을 만들고, 전달하여, 주행과 작업으로 이어집니다.

생성 → 전달 → 사용
유압
03

오일의 흐름과 압력으로 장치를 움직이고 힘을 제어합니다.

생성 → 전달 → 사용
조향
04

운전자의 입력을 방향 변화로 바꾸는 흐름입니다.

생성 → 전달 → 사용
CURRENT SYSTEM

전기

기계를 깨우고 시동하며 각 장치를 제어하는 기본 시스템입니다.

전기 → 생성 → 전달 → 사용

배경 실루엣을 사용하려면 assets/images/tractor-silhouette.png 파일을 준비해 넣으면 됩니다.

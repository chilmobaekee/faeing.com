상태점검

지금 보이는 증상과 느낌을 기준으로
먼저 확인해야 할 항목을 순서대로 제시합니다.
“감”이 아니라 “체크”로 시작합니다.

예: 시동 / 출력 / 소음 / 연기 / 유압 / PTO / 전기 / 경고등

구조원리

왜 이런 증상이 생기는지,
핵심 부품과 흐름을 “쉽게” 풀어서 설명합니다.
이해가 되면 두려움이 줄어듭니다.

예: 흡기-연료-연소 / 냉각 / 윤활 / 유압 / 전기-센서

정기관리

기계의 수명은 결국 습관입니다.
닦고, 조이고, 기름치고…
작은 루틴이 큰 고장을 막습니다.

예: 엔진오일 / 에어클리너 / 냉각수 / 연료필터 / 그리스 / 점검주기

보조도구

작업이 쉬워지는 도구는
실수를 줄이고 품질을 올립니다.
‘현장에서 바로 쓰는 것’만 남깁니다.

예: 레이저 정렬기 / 진단기 / 토크렌치 / 압력게이지 / 간단한 계측

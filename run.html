핵심 기능

복잡하게 만들지 않고, 꼭 필요한 것만 남겼다.

속도 유지

사용자가 설정한 목표 속도를 유지해 페이스가 흔들리지 않게.

  • A/B/C 프리셋
  • 미세 가감속
  • 정지/재시작

직관 조작

러닝 중에도 한 손으로 간단히 조작 가능.

  • 버튼 4개 설계
  • 손에 익는 UI
  • 상태 표시

안전 확장

환경에 따라 센서·표시부를 추가할 수 있게.

  • 초음파/라이다 옵션
  • 표시부(세그먼트) 옵션
  • 야간 LED 옵션

사용 흐름

“설치 → 설정 → 출발” 3단계면 끝.

1
설치

바퀴/전원/리모컨만 연결. 복잡한 세팅 최소화.

2
설정

A/B/C 프리셋 또는 +/−로 속도 세팅.

3
출발

START 누르면 일정 페이스로 전진. STOP은 즉시 정지.

키트 구성(예시)

프로토타입 기준. 향후 목적에 맞춰 모듈형으로 확장.

기본

  • 구동부(모터/휠/샤시)
  • ESP32 제어
  • 리모컨(START/STOP/UP/DOWN)
  • 전원(배터리 + DC-DC)

옵션

  • 장애물 감지(초음파/라이다)
  • 속도 표시(세그먼트)
  • 야간 가시 LED
  • 방수/방진 하우징

RUN을 “메인”처럼 단정하게.

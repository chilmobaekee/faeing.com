사용 흐름

설치 → 설정 → 출발, 세 단계면 충분합니다.

1
Setup

설치

전원과 구동부, 리모컨만 연결합니다.

  • 배터리 + DC-DC
  • 구동부 점검
  • 리모컨 페어링
2
Set

설정

프리셋 또는 UP/DOWN으로 목표 속도 설정.

  • A/B/C 프리셋
  • 미세 속도 조절
  • 표시부 확인
3
Go

출발

START로 출발, STOP은 즉시 정지.

  • 안정적 전진
  • 상황별 속도 변경
  • 즉시 정지
